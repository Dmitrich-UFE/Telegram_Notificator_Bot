# TelegramNotificatorBot

Бот, оповещающий пользователя в заданное пользователем время

## Usage

/start - начать общение 
/help - получить помощь 
/MyTasks - получить список действующих задач 
/createTask - создать задание. Запишите задачу, которую нужно напомнить, дату и готово! 

Если вам чужды такие команды, внизу будут кнопки
